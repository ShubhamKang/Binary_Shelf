
#📚 BinaryShelf – Tech Library

BinaryShelf is a simple and intuitive online catalog designed to help users organize, browse, and manage technical books and technology modules — all in one place.

🚀 Live Site

➡️ https://binarytechshelf.netlify.app/

📌 Features

Dashboard Overview
A clean dashboard that shows your current library status at a glance. 
binarytechshelf.netlify.app

Tech Catalog
Explore and manage a curated list of tech books and modules. 
binarytechshelf.netlify.app

Book Registry
Track book details such as Title, Author, ISBN, Status, and manage them with available actions. 
binarytechshelf.netlify.app

Module Catalog
A space for technological frameworks and language documentation.

🛠️ Getting Started

To run this project locally:

Clone the repository

git clone https://github.com/<your-username>/<repo-name>.git


Install Dependencies

npm install


Start Development Server

npm start


This launches the app locally (e.g., http://localhost:3000).
